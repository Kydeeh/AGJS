# AGJS飯店網站
TGA102 第四組專題

# 目錄說明
* agjs/bean：PO、VO
* agjs/common：spring組態、控制器、Jedis
* agjs/controller：Controller
* agjs/dao：DAO
* agjs/ecpay/payment/integration/：綠界API
* agjs/service：Service
* webapp/admin：後台前端(html、css、js)
* webapp/main：前台前端(html、css、js)

# 功能分類
* announcement：公告
* customer：客服
* journey：行程
* order：訂單
* restaurant：餐廳
* room：房間
* user：會員、管理員

# 使用技術
### 後端
* Java
* Spring MVC
* Hibernate
* Servlet
* JDBC

### 資料庫
* MySQL
* Redis

### 前端
* AJAX
* jQuery
* HTML
* CSS
